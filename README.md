# SOLO&CO

> AI 기반 맞춤형 여행 프로젝트 관리 플랫폼  
> _솔로·그룹 여행을 구분한 일정 자동 생성, 경로 추천, 커뮤니티 기반 여행 서비스_  
> <br/>

## 🔍 프로젝트 개요

<img width="1920" height="804" alt="image" src="https://github.com/user-attachments/assets/24b6b96a-c1c9-4a5c-b286-43c7a337f22a" />

**SOLO&CO**는 사용자가 여행 프로젝트를 생성하고 가고 싶은 장소를 담으면,  
**AI가 이동 동선과 여행 타입을 고려해 최적의 여행 일정을 자동으로 완성해주는 웹 서비스**입니다.

여행 방식을 **솔로 여행 / 그룹 여행**으로 분리하여  
혼자 여행하는 사용자를 위한 **안전·난이도 분석**,  
함께 여행하는 사용자를 위한 **초대·투표·커뮤니티 기능**을 제공함으로써  
실제 여행 준비 과정에서 발생하는 번거로움을 최소화했습니다.

<br/>

## 📈 기대 효과

- **여행 준비 시간 단축**: 장소 선택만으로 일정 자동 완성
- **의사결정 비용 감소**: 그룹 여행 시 투표·커뮤니티로 합의 과정 단순화
- **솔로 여행 안정성 강화**: 난이도 점수화로 사전 위험 요소 인지 가능
- **확장성 확보**: 여행 데이터 기반 추천 고도화 및 다양한 여행 타입 확장 가능

<br/>

## 🤖 기술 스택

- **Frontend**: Vue.js, Tailwind CSS
- **상태 관리·비동기**: Axios
- **지도/위치**: Google Maps API
- **AI 연동**: LLM(ChatGPT), JSON 기반 응답 구조화
- **인증**: JWT
- **배포**: Netlify
- **협업/설계**: Figma

<br/>

## 🧪 프로젝트 회고

- **AI 기능을 실제 UX에 녹여낸 경험**: 단순 추천이 아닌, 사용자가 신뢰하고 판단할 수 있는 점수·시각화 제공
- **솔로/그룹 여행 분리 설계**: 사용자 상황에 따라 다른 문제를 해결하는 구조 설계
- **스토리북 도입**: 컴포넌트 단위 개발로 UI 일관성 및 유지보수성 향상
- **기능 중심 구조 설계**: 프로젝트 확장성을 고려한 화면·데이터 흐름 구성
- **실사용 관점 개선 반복**: 시연 및 테스트 과정에서 UX 피드백 반영
